# 高清国产视频 静态电影网站

已生成纯静态 HTML 电影网站。

- 影片数据：2000 条
- 独立详情页：2000 个，位于 `movies/` 目录
- 分类页：10 个，位于网站根目录
- 首页：`index.html`
- 分类总览：`categories.html`
- 排行榜：`ranking.html`
- 全部影片搜索：`search.html`
- 播放器：详情页内点击“立即播放”后会加载本地 HLS 初始化脚本并绑定 m3u8 播放源

封面图片按规则引用网站顶级目录下的 `1.jpg` 到 `150.jpg`。上传素材中未包含这些图片文件，因此部署时请将对应 JPG 图片放到网站根目录，现有 HTML 已经按影片顺序写好引用路径。
